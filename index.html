About VS Code & Console Below

To set VS Code you have to create an empty folder, when on VS Code create index.html file or if you want JS/CSS files, then link them.

The console is a place where outputs and errors as well a degbugging help is displayed. To code in the console you need to use the script tag and use the console.log tag to code the console. To enter the console right click and then press inspect and press console.
